# Computer Port IT Solutions - Modern Web Portal

This repository contains the modernized, high-performance web portal for **Computer Port IT Solutions**, migrated from a legacy WordPress/Beaver Builder setup to a modern **React 18 + Vite** Single Page Application (SPA).

## 🚀 Live Demo
The project is configured for one-click deployment on **Vercel**.

## ✨ Key Features
- **Modern UI/UX**: Cyber-IT aesthetic with dark mode and glassmorphism.
- **Enterprise Routing**: Smooth, sub-second page transitions using `react-router-dom`.
- **Proxmox Comparison Tool**: Interactive side-by-side analysis of Proxmox vs VMware, Hyper-V, and Nutanix.
- **AI Infrastructure Assistant**: Floating conversational UI for automated client engagement.
- **Responsive Design**: Mobile-first architecture with a dedicated slide-in navigation drawer.
- **SEO Optimized**: Meta tags and structured data for IT service visibility.

## 🛠️ Tech Stack
- **Frontend**: React 18, Vite
- **Routing**: React Router 7
- **Styling**: Vanilla CSS (Custom Utility System)
- **Deployment**: Vercel ready (via `vercel.json`)

## 📦 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production
```bash
npm run build
```

## 🌐 Deployment on Vercel
This project is pre-configured with `vercel.json` to handle SPA routing.
1. Push this code to your GitHub repository.
2. Connect the repository to Vercel.
3. Vercel will automatically detect Vite and deploy your site.
